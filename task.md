# Помощник по уходу за растениями (0,8-1,5)

## Краткое описание:
Вам необходимо разработать веб-приложение по уходу за растениями. Система должна помогать новичкам понимать, какой уход требуется конкретному растению, а опытным цветоводам быстрее находить рекомендации, вести личный список растений и отслеживать задачи по уходу.

## Предполагаемый продукт:
Продукт должен состоять из веб-клиента, работающего на стороне пользователя. В базовом варианте данные можно хранить локально в браузере или внутри приложения. В системе должен быть справочник растений с рекомендациями по уходу, возможность добавлять растения в личный список, избранное и уведомления о необходимых действиях.

В справочнике растений должны быть карточки с описанием ухода. Пользователь должен иметь возможность открыть карточку растения и понять, как часто его нужно поливать, когда требуется пересадка, какое освещение ему подходит и есть ли особенности, которые важно учитывать при содержании дома.
Карточка растения должна включать: 
Название растения;
Рекомендации по поливу;
Рекомендации по освещению;
Информацию о пересадке;
Информацию о ядовитости;
Дополнительные особенности ухода, если они указаны в базе.

В базовой версии пользователь может добавлять растения в избранное и формировать личный список имеющихся растений, чтобы отслеживать уход именно за своей коллекцией, не требуя создания аккаунта. Для растений из личного списка пользователь должен иметь возможность указать информацию, которая поможет отслеживать уход: ссылку на соответствующую карточку из справочника, дату добавления растения, заметки и параметры напоминаний.

В приложении должны быть уведомления о необходимости полива или пересадки. Реализация уведомлений может зависеть от выбранной платформы, но в базовом сценарии пользователь должен получать напоминания о важных действиях по уходу за растениями из личного списка.

## Усложнения:
В дополнение к приложению реализуется серверную часть и хранилище данных, чтобы пользовательские списки, избранное и настройки напоминаний сохранялись между устройствами, а справочник был унифицирован в общей БД.
При реализованной серверной части добавляется обмен растениями между пользователями: размещение растения для обмена, указание предпочтений и чат для обсуждения условий.
Добавляется рекомендательная система, которая будет предлагать растения исходя из предпочтений пользователя и имеющейся коллекции в личном списке.
Реализуется распознавание растений по фотографии. Пользователь сможет загрузить изображение, а система попробует определить растение и показать подходящую карточку с возможностью добавления в личный список.
