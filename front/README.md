# Plant Helper — Frontend

Vanilla JS / HTML / CSS. Веб-клиент.

Структуру внутри команда фронта организует сама. Единственная жёсткая договорённость — **работать по API-контракту из корневого `../plan.md`**.

## Стыковка с бэком
- **Как запустить бэк локально (в т.ч. на Windows):** см. `../back/README.md` → раздел «Для фронтендеров»
- Базовый URL API: `http://localhost:3000/api`
- После login/register сохранить `token` в `localStorage` и слать в защищённых запросах:
  `Authorization: Bearer <token>`
- Справочник (`GET /api/plants`) доступен без токена.

Если чего-то не хватает в контракте — сказать бэку, добавим/поправим в `plan.md` совместно.
